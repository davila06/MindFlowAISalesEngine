# Docs Index

Esta carpeta concentra la documentación de referencia del proyecto MindFlow AI Sales Engine.

## Estructura

- `docs/architecture/`
  - Documentos de arquitectura por capa (backend, frontend, documentación, infra/devops) y principios estructurales.

- `docs/api/`
  - Artefactos OpenAPI versionados por release y guía de publicación.

- `docs/operations/`
  - Runbooks, playbooks de incidentes, políticas operativas, RBAC y auditoría doc-código.

- `docs/engineering/`
  - Estándares técnicos de codificación y prácticas de ingeniería.

- `docs/product/`
  - Documento maestro de producto, épicas funcionales, KPI dashboard y Definition of Done.

## Objetivo

Mantener la raíz del repositorio limpia y agrupar la documentación por propósito para mejorar mantenibilidad, trazabilidad y onboarding.
